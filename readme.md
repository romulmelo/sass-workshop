# Sass Workshop :paintbrush:
Instructional guide for the sass workshop for interns at @valtech.

## Requirements :brain:
To proceed with the workshop you need to know html, semantics and basic css.

### Tools :hammer_and_wrench:
It's necessary that you have on your machine:

- [Visual Studio Code](https://code.visualstudio.com/download)
- [Node](https://nodejs.org/en/) *LTS version.*
- [Yarn](https://yarnpkg.com/) (optional)


## Schedule :white_check_mark:
You can access the workshop schedule by accessing this [Notion Board](https://www.notion.so/Workshop-Criando-interfaces-futur-sticas-usando-Sass-88ab087cea9240f8acceb12313abe02f) :arrow_upper_right:

## How to execute :running_woman:

First, install dependencies using command

```bash
yarn

#or

npm install
```

For development mode, open terminal in your vscode and type

```bash
yarn dev

# or

npm run dev
```

Very easy, isn't it? You can now view the running project by accessing http://localhost:8080 :globe_with_meridians:

For build:

```bash
yarn build

# or

npm run build
```
